  • How does the Presale work?

    The Presale allows core investors to get first dibs on the new token. The funds will be used to create the Main LP and Marketing needs. The presale will be on a timer count down. All tokens bought during this time will be locked until the timer ends, then users are allowed to claim their tokens.

  • What is the Mint?

    Many users have asked for the ability to mint tokens, so we have created a paid token Minter. This token mint is special it will allow a user to purchase tokens at a set mint price, but as tokens are minted the price will increase by 3% for every 100,000 tokens minted. The funds are then used to do a buy back and burn of the supply from the Main LP.
